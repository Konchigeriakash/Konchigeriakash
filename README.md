### 🌾 Parvati Shankar Cold Pressed Edible Oil Unit — E-Commerce Platform
**A real, live, full-stack e-commerce app built for an actual small business.**
Customers browse cold-pressed oils, sign up/log in with JWT auth, manage a cart, checkout, and confirm delivery straight over WhatsApp via an auto-generated `wa.me` link. Ships with a role-based admin dashboard for managing products, orders, and sales stats.

`React 19` `Spring Boot 3` `Spring Security` `PostgreSQL` `JWT` &nbsp;·&nbsp; deployed on **Vercel + Render + Neon**
